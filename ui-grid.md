# Angular UI grid

[Angular UI grid](http://ui-grid.info/) is a datagrid fully integrated with AngularJS. It is part of the 
[AngularUI](http://angular-ui.github.io/) suite.

## Configuration

```
"bower_components/w20-components/w20-components.w20.json": {
    "grid": {}
}
```
